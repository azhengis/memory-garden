# Memory Garden

Memory Garden is a simple iOS app built with Swift. It allows users to store personal memories and view them in an organized and visual way.

## Features

Users can add and save memories  
Memories are displayed in a clean interface  
Data is stored locally on the device  

## Tech Stack

Swift  
UIKit / SwiftUI  
Local storage  

## Purpose

This project was built to practice iOS development and working with data storage and user interfaces.

## How to Run

Clone the repository  
Open the project in Xcode  
Run the app on a simulator or device
